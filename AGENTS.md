# AGENTS.md

Build for mobile first, especially iPhone Safari.

Project priorities:
1. Speed over precision
2. Simplicity over feature breadth
3. Memorability of the mental formula
4. Clean, minimal UX

Technical rules:
- Use Next.js + TypeScript + Tailwind
- Keep components small and reusable
- Keep formula logic separate from UI
- Avoid unnecessary dependencies
- Prefer readable code over clever code
- Preserve a simple architecture

UX rules:
- Single-column layouts
- Large touch targets
- No hover-dependent interactions
- High contrast and clear typography
- Keep copy plain and concise

Product rules:
- This is a shopping estimation tool, not a finance tool
- Always prefer a simpler formula when accuracy is close
- Optimize formulas for common shopping amounts
- Show exact conversion only as a secondary reference

When making changes:
- summarize what changed
- list files touched
- explain how to test
- note assumptions and open issues
